Workshop Overview

Introduction

Assistive robot technology allows humans and robots to collaborate on complex physical tasks, with applications ranging from manufacturing to healthcare to military. However, many assistive robots are either fully teleoperated---in which the user remotely controls all of the robot’s motion---or fully autonomous---in which a user may specify a goal, but the robot plans and executes its actions without additional input. Shared autonomy blends these two extremes by combining user input with autonomous robot assistance. Given a user’s control of the robot through input devices such as a joystick or a brain-computer interface, the system uses this input to reason about the user’s intentions, then executes actions that satisfy those intentions. The robot continually updates its user model while performing actions, allowing for seamless increases or decreases of autonomous assistance based on an adapting understanding of the user. In this workshop, we aim to advance the state of the art in shared autonomy by inviting research across a variety of application domains and areas, all with the goal of understanding how to successfully combine human control and robot autonomy for assistive systems.

The Shared Autonomy Workshop will be held at RSS 2016 in Ann Arbor, MI on June 18 or 19, 2016. The workshop will bring together researchers working broadly toward safe, effective, and intuitive mechanisms for shared control of robotic systems.

Topics include (but are not limited to):

  • user modeling and detecting user preferences
  • recognizing task activities from behavior
  • inverse reinforcement learning and inverse optimal control
  • real-time adaptation of models
  • human-robot interaction and collaboration
  • brain-computer interfaces
  • robot control input devices and interfaces for users with disabilities
  • shared control in the domains of: rehabilitation, assistive devices, remote presence, elder care, manufacturing, space, bomb disposal, search and rescue, construction, etc.
